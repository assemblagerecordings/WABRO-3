# WABRO X ASSEMBLAGE MASTER SPEC

## Summary

WABRO X ASSEMBLAGE is a state-responsive audiovisual instrument and creative ecosystem.

It translates cognitive and emotional state data into evolving neurofunk, visual systems, performance logic, and eventually collaborative creative infrastructure.

The goal is not mind-reading.

The goal is state-responsive creativity.

## Core Thesis

Traditional instruments translate physical gestures into sound.

WABRO X ASSEMBLAGE translates performer state into evolving audiovisual systems.

The performer does not directly play individual notes. The performer influences higher-level states that reshape snare architecture, bass pressure, spectral movement, arrangement behaviour, and visual form.

## Core Concepts

### Signal vs Noise

Signal and noise are musical, technical, and philosophical materials.

Musically:
- snare fundamentals
- white noise
- spectral movement

Technically:
- EEG streams
- artifacts
- smoothing
- state inference

Artistically:
- intention
- randomness
- emergence

### Assemblage

Creative systems are built from interacting parts:

- performer
- EEG data
- Bitwig project
- snare layers
- spectral engines
- visual systems
- AI collaborators
- audience feedback
